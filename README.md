splitImage
==========

##实现效果

用户可以通过上传、拖拽图片，自定义切片个数实现分割图片，并可以保存分割后的图片。

##兼容性

因为使用到HTML5的 Drag And Drop、FILE API，canvas等技术，所以支持的浏览器有限，尤其是IE。。。自己可以测试的浏览器有限，因为电脑上的浏览器都是最新版本的

所以，支持的浏览器有IE 10+、chrome 30、FF 24、opera 17，其他的待测。。。
